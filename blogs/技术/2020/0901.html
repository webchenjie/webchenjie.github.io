    # 分片上传

    1、把 file 大小等信息传递给后端，后端会返回每次上传的分片大小等信息
2、前端根据后端的大小使用 file.slice 分割 file 进行上传
3、每个分片 file 上传成功后会返回当前还剩的 file 信息
4、如果判断到已上传完成，则前端需要通知后端已上传完成，后端会合成文件
