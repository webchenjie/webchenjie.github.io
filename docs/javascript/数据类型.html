数据类型

chenjie 2020-07-04 
// 简单数据类型（不可改变，只能复制，按值传递）
Symbol
undefined
Boolean
Number
String
1. 值不可修改
2. 保存在栈内存中
3. 按值访问
4. 比较时值相等即相等
5. 复制时创建一个副本
6. 按值传递参数
7. 用typeof检测类型
----------------------------------------------------------------------------------------------
// 复杂数据类型（引用，可以改变，按引用传递）
Object
Array
Function
null
1. 值可以修改
2. 保存在堆内存中
3. 按引用访问
4. 比较时同一引用才相等
5. 复制其实是指针
6. 按值传递参数
7. 用instanceof检测类型
----------------------------------------------------------------------------------------------
// 简单数据类型检测,返回值String, Number, Boolean, Object, undefined, function
typeOf(1) // 返回Numbner
null === Object // true
undefined == null // true, undefined是null派生出来的
NaN // 非数值
NaN !== NaN // true
isNaN(参数) // 判断参数是否是非数值，返回Boolean，会尝试把参数转为数字类型如 id = '16'
----------------------------------------------------------------------------------------------
// 复杂数据类型检验，typeof遇到复杂数据类型只会返回object
[] instanceof Array // 返回true
----------------------------------------------------------------------------------------------
// 数值转换
Number(参数) // 参数要以数字开头， 会忽略开头0和空格，如果转换字符串则返回NaN，有两个参数的第二个为进制
parsetInt(参数) // 参数要以数字开头， 会忽略开头0和空格，如果转换字符串则返回NaN，有两个参数的第二个为进制
parseFloat(参数) // 参数要以数字开头， 会忽略开头0和空格，如果转换字符串则返回NaN，有两个参数的第二个为进制，第一个小数点有效
----------------------------------------------------------------------------------------------
// 字符串转换
String(参数) // 在不知道参数的类型时使用
toStirng() // 返回str的一个副本
----------------------------------------------------------------------------------------------
// boolean转换
1. 除0之外的所有数字转换为布尔值都为true
2. 除空之外的所有字符串转换为布尔值都为true
3. null和undefined转换为布尔值为false
4. 比较操作符返回值为boolean
----------------------------------------------------------------------------------------------
// 逻辑与操作符转换
逻辑与（只要有一个条件不成立则返回false）
如果操作数不是布尔值时：
1. 如果第一个操作数隐式类型转换后为true则返回最后一个操作数
2. 如果第一个操作数隐式类型转换后为false则返回第一个操作数
3. 有一个操作数是null则返回null
4. 有一个操作数是undefined则返回undefined
----------------------------------------------------------------------------------------------
// 逻辑或操作符转换
逻辑或（只要有一个条件成立则返回true）
如果操作数不是布尔值时：
1. 如果第一个操作数隐式类型转换后为true则返回第一个操作数
2. 如果第一个操作数隐式类型转换后为false则返回第二个操作数
3. 如果有两个操作数是null则返回null
4. 如果有两个操作数是undefined则返回undefined
5. 如果有两个操作数是NaN则返回NaN
----------------------------------------------------------------------------------------------
// 逻辑非操作符转换
1. 无论操作数是什么数据类型，都会返回一个布尔值
2. !!对该布尔值求反
