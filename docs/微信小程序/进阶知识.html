// зҷ»еҪ•,иҺ·еҸ–з”ЁжҲ·дҝЎжҒҜ
1. wx.login // дёҚйңҖиҰҒжҺҲжқғ,йқҷй»ҳзҷ»еҪ•,зҷ»еҪ•зҠ¶жҖҒжңүж—¶ж•ҲжҖ§,еӨҡд№…еӨұж•Ҳз”ұеҫ®дҝЎжңҚеҠЎеҷЁеҶіе®ҡ
2. wx.openSetting() // з”ЁжҲ·е…ЁйғЁзҡ„жҺҲжқғи®°еҪ•,и®©з”ЁжҲ·йҮҚж–°жҺҲжқғ
3. wx.getUserInfo
    иҺ·еҸ–з”ЁжҲ·дҝЎжҒҜ(жҳҺж–ҮгҖҒеҠ еҜҶ),йңҖиҰҒз”ЁжҲ·жҺҲжқғ
    жңүwichCredentialsеҸӮж•°,еҰӮжһңдёәtrueж—¶,еҝ…йЎ»иҰҒжұӮз”ЁжҲ·е·Ізҷ»еҪ•еҗҰеҲҷжӢҝдёҚеҲ°дҝЎжҒҜ
    дёәfalseж—¶,дёҚйңҖиҰҒзҷ»еҪ•,дҪҶжӢҝдёҚеҲ°еҠ еҜҶдҝЎжҒҜ,еҰӮжһңжҺҲжқғиў«жӢ’з»қеҲҷдёҚдјҡеҶҚжҳҫзӨәи§ҰеҸ‘жҺҲжқғ
    еҰӮжһңиҰҒйҮҚж–°и§ҰеҸ‘еҲҷйңҖи°ғз”Ёwx.openSetting()
4. uniIDйңҖиҰҒжҺҲжқғеҗҺжӢҝеҲ°,openIDдёҚз”Ё,еңЁеҫ®дҝЎе…¬дј—е№іеҸ°дёӢ,openIDдёҚдёҖж ·,uniIDдёҖж ·
5. еңЁж–°зүҲзҡ„е°ҸзЁӢеәҸдёӯиҰҒжғіи®©з”ЁжҲ·жҺҲжқғеҝ…йЎ»дҪҝз”Ёbuttonи®©з”ЁжҲ·зӮ№еҮ»еј•еҜјжҺҲжқғ
   йҖҡиҝҮwx.getSettingеҲӨж–ӯз”ЁжҲ·жҳҜеҗҰжҺҲжқғ,еҰӮжһңз”ЁжҲ·е·Із»ҸжҺҲжқғ,еҸҜд»Ҙи°ғз”ЁgetUserInfoиҺ·еҸ–з”ЁжҲ·дҝЎжҒҜ,жІЎжҺҲжқғеҲҷиҺ·еҸ–дёҚеҲ°
6. еңЁдҪҝз”ЁжҺҲжқғbuttonж—¶<button open-type="getUserInfo" bind:getUserInfo="ж–№жі•еҗҚ"> // йҖҡиҝҮж–№жі•еҗҚзҡ„eventжӢҝеҲ°з”ЁжҲ·дҝЎжҒҜ
7. еҶ…зҪ®з»„д»¶<open-data>иҝҷдёӘеҸҜд»ҘиҺ·еҸ–з”ЁжҲ·дҝЎжҒҜ,еҸӘеңЁwxmlдёӯжңүз”Ё

// з»„д»¶
1. дҪҝз”Ёз»„д»¶ж—¶йңҖиҰҒеңЁдҪҝз”Ёзҡ„йЎөйқўзҡ„jsonдёӯй…ҚзҪ®еј•е…Ҙ:
    "usingComponents": { 'з»„д»¶еҗҚ': 'з»„д»¶зӣ®еҪ•' }
    еҚіеҸҜеңЁwxmlдёӯдҪҝз”Ё<з»„д»¶еҗҚ></з»„д»¶еҗҚ>
2. з»„д»¶еҸӘиғҪз»§жүҝе…ЁеұҖж–Үд»¶зҡ„font/colorзҡ„css
3. е°ҸзЁӢеәҸеӯҗз»„д»¶еҗ‘зҲ¶з»„д»¶дј еҖјж—¶зҡ„this.triggerEventдёҺvueзҡ„this.$emitдёҖж ·
4. дёҚиҰҒеңЁpropertiesзҡ„observerдёӯдҝ®ж”№иҮӘиә«зҡ„еҖј,дјҡи§ҰеҸ‘ж— йҷҗйҖ’еҪ’ // this.properties.ж•°жҚ®еҗҚ, observerзӣёеҪ“дәҺvueзҡ„watch
5. е°ҸзЁӢеәҸиҮӘе®ҡд№үз»„д»¶зҡ„жү©еұ•behaviorжҳҜз»„д»¶дёӯjsзҡ„жҠҪиұЎ,жҠҠз»„д»¶дёӯзӣёеҗҢзҡ„дёңиҘҝжҸҗеҸ–еҲ°дёҖдёӘbehaviorдёӯ,дҪҝз”ЁдёҺиҮӘе®ҡд№үз»„д»¶дёҖж ·
   йңҖиҰҒз»„д»¶еҜје…ҘгҖҒжіЁеҶҢ,еңЁеұһжҖ§ж–№йқў,з»„д»¶дёӯдёҺbehaviorеҸ‘з”ҹйҮҚеҗҚж—¶дјҡиҰҶзӣ–,еҶҷеңЁbehaviorеҗҺйқў(еӨҡдёӘ)дјҡиҰҶзӣ–еүҚйқўзҡ„
   иҖҢз”ҹе‘Ҫе‘ЁжңҹдёҚдјҡиҰҶзӣ–,дјҡдҫқж¬Ўжү§иЎҢ
6. еңЁpropertiesдёӯз»‘е®ҡзҡ„ж•°жҚ®еңЁwxmlдёӯеҸҜд»ҘзӣҙжҺҘз”ЁеұһжҖ§зҡ„ж–№ејҸеңЁз»„д»¶дёӯдҪҝз”Ё
7. з»„д»¶зҡ„з”ҹе‘Ҫе‘Ёжңҹ:
    created() // жёІжҹ“еүҚ
    attached() // жёІжҹ“ж—¶
    ready() // жёІжҹ“еҗҺ
    mored() // з»„д»¶з§»еҠЁж—¶
    datached() // з»„д»¶еҲ йҷӨж—¶
8. еңЁз»„д»¶дёӯзҡ„this.dataе’Ңthis.propertiesжҢҮзҡ„жҳҜеҗҢдёҖдёӘjsеҜ№иұЎ
9. иҮӘе®ҡд№үз»„д»¶дёҚж”ҜжҢҒhiddenеұһжҖ§,еҰӮиҰҒе®һзҺ°еҲҷеңЁз»„д»¶зҡ„propertiesдёӯж·»еҠ hiddenеұһжҖ§,з„¶еҗҺеңЁз»„д»¶зҡ„wxmlдёӯз»‘е®ҡеұһжҖ§
10. з»„д»¶дёӯеҸӘиҝӣиЎҢиҮӘиә«зҡ„йҖ»иҫ‘,е…ідәҺе…¶д»–зҡ„дёҡеҠЎеҲҷж”ҫеңЁи°ғз”Ёж–№иҝӣиЎҢ // зІ’еәҰ
11. еңЁе°ҸзЁӢеәҸзҡ„з»„д»¶дёӯдҪҝз”ЁжҸ’ж§Ҫslotе’ҢvueдёӯдёҖж ·,дҪҶйңҖиҰҒеңЁз»„д»¶зҡ„jsдёӯзҡ„componentдёӢж–°еўһoptions: { multipleslots: true }жүҚиғҪдҪҝз”Ё
12. дёҚиғҪеҜ№иҮӘе®ҡд№үеҗҚеӯ—зҡ„з»„д»¶иҝӣиЎҢcssдҝ®йҘ°ж— ж•Ҳ,еҸҜд»ҘеҜ№иҮӘе®ҡд№үз»„д»¶дёӢзҡ„жҹҗдёҖдёӘж Үзӯҫдҝ®йҘ°
13. з»„д»¶зҡ„cssеҸҜд»ҘйҖҡиҝҮе°ҸзЁӢеәҸзҡ„еӨ–йғЁж ·ејҸжқҘж”№еҸҳеҶ…йғЁж ·ејҸ
    еңЁcomponentдёӢж–°еўһextermalclasses: ['ж ·ејҸеҗҚ'],з„¶еҗҺжҠҠж ·ејҸеҗҚеҠ еңЁйңҖиҰҒеә”з”Ёзҡ„ж ҮзӯҫдёҠ
    еңЁеӨ–йғЁзҡ„йЎөйқўдёӯзҡ„з»„д»¶еҗҚдёҠеўһеҠ  ж ·ејҸеҗҚ="иҮӘе®ҡд№үж ·ејҸеҗҚ",дёҖдёӘжҳҜеҶ…йғЁж ·ејҸеҗҚ,дёҖдёӘжҳҜеӨ–йғЁж ·ејҸеҗҚ
    йңҖиҰҒеҠ !important,д№ҹеҸҜжҸҗеҚҮдјҳе…Ҳзә§зҡ„е…¶д»–ж–№жі•,й»ҳи®ӨдёҚзЎ®е®ҡеӨ–йғЁдјҡиҰҶзӣ–еҶ…йғЁ
14. е°ҸзЁӢеәҸзҡ„{{}}еҸӘиғҪеҶҷз®ҖеҚ•зҡ„jsиЎЁиҫҫејҸ,иҖҢдёҚиғҪи°ғз”Ёж–№жі•
15. wxsеҸҜд»ҘеңЁwxmlдёӯи°ғз”Ёjsе’Ңes6зҡ„jsдёҚдёҖж ·
    еңЁwxmlдёӯйҖҡиҝҮ<wxs src="wxsж–Үд»¶зҡ„и·Ҝеҫ„" module="иҮӘе®ҡд№үеҗҚеӯ—">
    еңЁwxmlдёӯйҖҡиҝҮиҮӘе®ҡд№үеҗҚеӯ—.еҮҪж•°еҗҚи°ғз”Ё
    еңЁwxsдёӯйҖҡиҝҮmodule.export = { еҮҪж•°еҗҚ }еҜјеҮә
    д№ҹеҸҜзӣҙжҺҘеңЁwxsдёӯеөҢе…Ҙwxsд»Јз Ғ<wxs module="иҮӘе®ҡд№ү">еҶ…е®№</wxs>
16. з»„д»¶д№ҹеҸҜд»Ҙеј•з”Ёз»„д»¶гҖҒж–№жі•е’ҢйЎөйқўеј•з”Ёз»„д»¶дёҖж ·
----------------------------------------------------------------------------------------------
1. еңЁjsжӢҝеҲ°ж•°жҚ®еҗҺйҖҡиҝҮsetDataз»‘е®ҡеҲ°dataдёӯ,еҚіеҸҜеңЁwxmlдёӯдҪҝз”Ё
2. wx:keyзҡ„еҸ–еҖјдёәдёҚйҮҚеӨҚзҡ„ж•°еӯ—жҲ–еӯ—з¬ҰдёІ,еҰӮжһңйҒҚеҺҶдёӢзҡ„еҲ—иЎЁжҳҜж•°еӯ—жҲ–еӯ—з¬ҰдёІ,еҸҜд»Ҙwx:key = "*this" // *this
3. йҖҡиҝҮй”Ғзҡ„жҰӮеҝөи§ЈеҶійҮҚеӨҚеҠ иҪҪж•°жҚ®зҡ„й—®йўҳ,еңЁеҠ иҪҪеүҚе®ҡд№үдёҖдёӘflgеҸҳйҮҸиҝӣиЎҢеҲӨж–ӯ,иҝӣиЎҢдёӯдёҚжү§иЎҢеҠ иҪҪд»Јз Ғ
4. this.setDataе’Ңthis.data.еҗҚеӯ—,еҰӮжһңж•°жҚ®еңЁwxmlдёӯжңүеә”з”Ёз»‘е®ҡеҲҷдҪҝз”Ёthis.setData,еҰӮжһңжІЎжңүеҸҜд»ҘдҪҝз”Ёthis.data.еҗҚеӯ— // еңЁз»„д»¶дёӯ
5. еҸҜд»ҘдҪҝз”ЁеӣҫзүҮжҢүй’®и®©жҢүй’®иҮӘе®ҡд№үж ·ејҸ
6. е…¬дј—еҸ·еҲҶдёәжңҚеҠЎеҸ·е’Ңи®ўйҳ…еҸ·
